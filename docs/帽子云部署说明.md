# 帽子云部署说明

本项目为纯静态 HTML，可直接在帽子云部署，无需构建。

## 前置条件

- 代码已推送到 **GitHub**（帽子云仅支持 GitHub 仓库）
- 本仓库：<https://github.com/FrankJIE09/gemini_htmls>

## 部署步骤

### 1. 注册 / 登录

打开：<https://dash.maoziyun.com/login>  
若无账号，先到 <https://dash.maoziyun.com/register> 注册。

### 2. 创建应用

在控制台点击「创建应用」或类似入口。

### 3. 授权 GitHub

- 选择「从 GitHub 部署」或「连接 GitHub」
- 按提示授权帽子云访问你的 GitHub 账号
- 授权后可看到仓库列表

### 4. 选择仓库与分支

- **仓库**：`FrankJIE09/gemini_htmls`
- **分支**：`main`

### 5. 构建配置（纯静态可留空）

- **构建命令**：留空（不填）
- **输出目录**：留空或填 `.`（表示仓库根目录即网站根目录）
- 若无「输出目录」选项，保持默认即可，系统会以根目录为静态站点

### 6. 创建并等待构建

点击「创建应用」，等待构建完成。  
完成后会得到一个类似 `xxx.maoziyun.com` 的访问地址，首页即为 `index.html`。

### 7. 后续更新

每次向 GitHub 的 `main` 分支 push 后，帽子云一般会自动重新部署（若已开启自动部署）。也可在控制台手动触发「重新部署」。

## 访问地址

部署成功后：

- 首页：`https://<分配的子域>.maoziyun.com/`
- 各工具页：`https://<分配的子域>.maoziyun.com/xxx.html`（如 `car_loan_calculator.html`）

## 自定义域名（可选）

在应用设置中可绑定自己的域名，并开启自动 HTTPS。
