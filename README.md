# gemini_htmls

一个包含多个实用计算器的 HTML 工具集合。

## 项目简介

本项目包含多个独立的 HTML 计算器工具，可以直接在浏览器中打开使用，无需安装任何依赖。

## 文件列表

### 1. 上海年薪计算器 (shanghai_salary_calculator.html)
- 功能：计算上海地区的年薪收入，使用累计预扣法
- 用途：帮助计算个人所得税和实际到手收入

### 2. 房贷计算器 (mortgage_calculator.html)
- 功能：计算房贷月供、总利息、还款计划等
- 用途：帮助规划房贷还款方案

### 3. 车贷计算器 (car_loan_calculator.html)
- 功能：计算车贷月供、总利息、还款明细等
- 用途：帮助规划车贷还款方案

## 使用方法

### 本地使用

直接在浏览器中打开对应的 HTML 文件即可使用，无需任何配置。

### 在线访问（GitHub Pages）

在 GitHub 仓库中开启 Pages 后，可通过 Pages 提供的链接在线访问。建议使用仓库根目录的 `index.html` 作为入口页。

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (React)
