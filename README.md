# gemini_htmls

一个包含多个实用计算器的 HTML 工具集合。

**在线访问：** https://frankjie09.github.io/gemini_htmls/

## 项目简介

本项目包含多个独立的 HTML 计算器工具，可以直接在浏览器中打开使用，无需安装任何依赖。

## 文件列表

### 1. 上海年薪计算器 (shanghai_salary_calculator.html)
- 功能：计算上海地区的年薪收入，使用累计预扣法
- 用途：帮助计算个人所得税和实际到手收入

### 2. 房贷计算器 (mortgage_calculator.html)
- 功能：计算房贷月供、总利息、还款计划等
- 用途：帮助规划房贷还款方案

### 3. 车贷计算器 (car_loan_calculator.html)
- 功能：计算车贷月供、总利息、还款明细等
- 用途：帮助规划车贷还款方案

### 4. 上海社保/养老金计算器 (shanghai_pension_calculator.html)
- 功能：计算上海社保与养老金相关金额
- 用途：帮助了解社保缴费与养老金预估

### 5. Mermaid 图表生成器 (Mermaid图表生成器.html)
- 功能：在线编辑并预览 Mermaid 图表
- 用途：快速绘制流程图、时序图等

### 6. 纳斯达克 vs 利率数据分析仪 (nasdaq_rates_analyzer.html)
- 功能：上传 CSV 数据，可视化纳斯达克指数与利率关系，支持日期范围滑块与统计信息
- 用途：分析股指与利率的历史走势与相关性

### 7. 赛博朋克粒子效果 (cyberpunk_particles*.html)
- 功能：Three.js 粒子动效；移动端版本支持摄像头交互
- 用途：展示/娱乐
